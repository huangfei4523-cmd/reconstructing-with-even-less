## 新增需求

### 需求: 力导向布局重建 2D 分布

Phase 3 必须从 Phase 2 输出的边集合 E_hat 中重建点在 2D 平面上的分布。

#### 场景: 建有权图

- **当** Phase 2 输出边集合 E_hat = {(i,j,prob)}
- **那么** 构建有权图 G，边权重 w_ij = prob
- **而且** 图中包含所有 N 个节点（含孤立点）

#### 场景: 弹簧-斥力能量最小化

- **当** 有权图 G 构建完毕
- **那么** 在图上运行弹簧-斥力模型收敛到稳态：
  - 弹簧力：相邻节点之间的引力 ∝ w_ij × |pos_i - pos_j|
  - 库仑力：所有节点对之间的斥力 ∝ 1/|pos_i - pos_j|
- **而且** 收敛条件为所有节点位移 < ε

#### 场景: 输出 2D 坐标

- **当** 力导向布局收敛
- **那么** 输出每个点的 2D 坐标 pos[N,2]
- **而且** 坐标尺度可任意（不求绝对位置正确，只求相对形状正确）

### 需求: 形状评估（可选）

如果攻击者有部分锚点信息，可以对重建结果进行 Procrustes 对齐和定量评估。

#### 场景: Procrustes 对齐

- **当** 攻击者已知少量点的真实坐标（如 3-5 个锚点）
- **那么** 用 Procrustes 变换将重建的 2D 坐标旋转、缩放、平移对齐到真实坐标系

#### 场景: Hausdorff 距离评估

- **当** 对齐后的重建坐标与真实坐标比较
- **那么** 计算两个点集之间的 Hausdorff 距离作为重建精度的度量

### 需求: 重建失败检测

Phase 3 必须能够检测重建是否失败（避免输出无意义的随机形状）。

#### 场景: 边数过少判定

- **当** Phase 2 输出的边数 < N/2
- **那么** 判定重建失败：边数不足以形成连通拓扑
- **而且** 报告 Phase 2 可能需要更多迭代或预训练不足

#### 场景: 布局离散度判定

- **当** 力导向布局收敛后所有点集中在极小区域内（方差 < 预期值）
- **那么** 判定重建失败：边未提供足够的空间约束
- **而且** 报告可能的原因：自训练未收敛或目标数据共现信号太弱
