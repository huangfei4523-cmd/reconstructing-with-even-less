## 新增需求

### 需求: Phase 2 自训练模块

`gnn_self_training.py` 必须实现设计文档详细设计 §2 定义的完整自训练流程。

#### 场景: 伪标签筛选

- **当** 调用 `SelectPseudoLabels(prob_matrix, N)`
- **那么** 返回 Top-2N 条最高概率边作为伪正样本、Bottom-10N 条最低概率边作为伪负样本
- **而且** 伪正样本必须覆盖 ≥80% 的节点
- **而且** 中间概率区域的对不参与训练

#### 场景: 迭代微调

- **当** 调用 `SelfTrainingLoop(model, C_target, max_iter=20)`
- **那么** 每轮在目标共现矩阵上推理 → 筛选伪标签 → 微调模型
- **而且** 微调损失包含一致性正则化项（随机屏蔽 5% 响应重算 C'，要求 GNN(C)≈GNN(C')）
- **而且** 只微调边预测器和最后一层消息传递参数（冻结编码器）

#### 场景: 收敛判定

- **当** 连续两轮自训练的伪正样本边集合重合率 ≥ 90%
- **那么** 自训练终止，输出推断边集合和微调后的模型

#### 场景: 发散检测

- **当** 连续 3 轮伪正样本边集合重合率持续下降
- **那么** 自训练终止，取历史最佳边集合并报告发散原因

### 需求: 输入输出约束

自训练模块必须只通过共现矩阵与外部交互。

#### 场景: 输入隔离

- **当** Phase 2 启动
- **那么** 输入仅包含 C_target[N,N] 矩阵 + 预训练模型参数
- **而且** 禁止 import 或使用 data_loader.py / process_database.py / 数据集特定逻辑

#### 场景: 输出格式

- **当** Phase 2 完成
- **那么** 输出 E_hat = [(i, j, prob), ...]，其中 prob 为 float ∈ [0,1]
