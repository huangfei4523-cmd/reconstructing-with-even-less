## 新增需求

### 需求: Phase 3 力导向布局

`gnn_reconstruction.py` 必须实现设计文档详细设计 §3 定义的形状重建流程。

#### 场景: 有权图构建与布局

- **当** 输入 E_hat = {(i,j,prob)}
- **那么** 构建有权图 G，边权重 = prob
- **而且** 在 G 上运行弹簧-斥力模型，输出 pos[N,2] 的 2D 坐标

#### 场景: 锚点支持

- **当** 提供可选锚点列表 anchors={(k, (x,y))}
- **那么** 锚点节点的位置固定，其余节点围绕锚点展开

#### 场景: 重建失败检测

- **当** 布局完成
- **那么** 检查: 边数 ≥ N/2、节点方差 ≥ 阈值、连通分量数 ≤ N/4
- **而且** 任何条件失败时报告具体原因

#### 场景: Procrustes 对齐

- **当** 提供锚点和真实坐标
- **那么** 计算最优旋转/缩放/平移，输出对齐后的坐标和 Hausdorff 距离
