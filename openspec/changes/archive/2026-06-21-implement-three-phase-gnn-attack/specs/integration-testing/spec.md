## 新增需求

### 需求: Phase 1 数据生成测试

`tests/test_phase1_data.py` 必须验证训练数据生成器对参数空间各维度组合的正确性。

#### 场景: 场景配置覆盖

- **当** 遍历 N={20,40,200}、形状={grid,random}、p={1%,50%}
- **那么** 每个组合都能成功生成有效样本（N≥4，有 >0 条正样本边）
- **而且** 生成时间 < 2 秒/样本

#### 场景: 特征维度正确

- **当** 生成任意配置的样本
- **那么** node_features.shape 为 [N,16]，adj_labels.shape 为 [N,N]

### 需求: Phase 2 伪标签测试

`tests/test_phase2_pseudo.py` 必须验证伪标签筛选的正确性。

#### 场景: 选取数量正确

- **当** 输入 N×N 概率矩阵和 N
- **那么** 伪正样本数 = 2N，伪负样本数 = 10N

#### 场景: 正样本覆盖度

- **当** 伪正样本数量 > 2N（有足够候选时）
- **那么** 选出的伪正样本覆盖 ≥ 80% 的节点

### 需求: Phase 3 布局测试

`tests/test_phase3_layout.py` 必须验证力导向布局的收敛性。

#### 场景: 简单图收敛

- **当** 输入一个连通网格图（N=25）
- **那么** 力导向布局在 < 100 次迭代内收敛
- **而且** 输出坐标非全零且方差 > 0

### 需求: 三阶段集成测试

`tests/test_integration.py` 必须验证端到端全流程。

#### 场景: cali_self 端到端

- **当** 在 cali_self（5点）数据集上运行三阶段全流程
- **那么** Phase 1 训练在 <30s 内完成（小规模快速验证）
- **而且** Phase 2 自训练在 <3 轮内收敛
- **而且** Phase 3 输出 5 个点的 2D 坐标
