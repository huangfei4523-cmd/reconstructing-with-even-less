### 需求: Phase 2 自训练模块

`gnn_self_training.py` 必须实现设计文档详细设计 §2 定义的完整自训练流程。

#### 场景: 伪标签筛选

- **当** 调用 `SelectPseudoLabels(prob_matrix, N)`
- **那么** 返回 Top-2N 条最高概率边作为伪正样本、Bottom-10N 条最低概率边作为伪负样本
- **而且** 伪正样本必须覆盖 ≥80% 的节点
- **而且** 中间概率区域的对不参与训练

#### 场景: 迭代微调

- **当** 调用 `SelfTrainingLoop(model, C_target, max_iter=20)`
- **那么** 每轮在目标共现矩阵上推理 → 筛选伪标签 → 微调模型
- **而且** 微调损失包含一致性正则化项（随机屏蔽 5% 响应重算 C'，要求 GNN(C)≈GNN(C')）
- **而且** 只微调边预测器和最后一层消息传递参数（冻结编码器）

#### 场景: 收敛判定

- **当** 连续两轮自训练的伪正样本边集合重合率 ≥ 90%
- **那么** 自训练终止，输出推断边集合和微调后的模型

#### 场景: 发散检测

- **当** 连续 3 轮伪正样本边集合重合率持续下降
- **那么** 自训练终止，取历史最佳边集合并报告发散原因

#### 场景: 微调消息传递使用完整图

- **当** `SelfTrainingLoop` 的每次微调迭代执行前向传播
- **那么** 消息传递的 `edge_index` 必须是完整的共现图（通过 `build_cooc_message_graph` 构建）
- **而且** 边预测 logits 只对伪标签边单独计算：拼接 `node_emb[src]` + `node_emb[dst]` → `edge_predictor`
- **而且** 不得只用伪标签边作为消息传递图

#### 场景: 一致性正则化 logits 长度一致

- **当** 计算扰动后共现的 consistency loss
- **那么** `logits_orig` 和 `logits_p` 必须使用相同的 `edge_idx_t` 做消息传递
- **那么** 只替换 `node_feat`（扰动前后），不替换 `edge_index`
- **那么** 无需 `min_len` 截断

### 需求: 输入输出约束

自训练模块必须只通过共现矩阵与外部交互。

#### 场景: 输入隔离

- **当** Phase 2 启动
- **那么** 输入仅包含 C_target[N,N] 矩阵 + 预训练模型参数
- **而且** 禁止 import 或使用 data_loader.py / process_database.py / 数据集特定逻辑

#### 场景: 输出格式

- **当** Phase 2 完成
- **那么** 输出 E_hat = [(i, j, prob), ...]，其中 prob 为 float ∈ [0,1]

### 需求: Phase 2 诊断日志

`SelfTrainingLoop` 必须输出诊断信息帮助判断模型学习状态。

#### 场景: 输出伪标签置信度

- **当** 每轮筛选伪标签后
- **那么** 打印伪正标签的平均预测概率、伪负标签的平均预测概率
- **而且** 打印全部 N(N-1)/2 个点对的概率分布熵（最大熵≈0.693 表示模型无区分能力）

#### 场景: 输出 C_target 特征统计

- **当** Phase 2 启动
- **那么** 打印 C_target 的关键统计：每行总强度均值/std、非零邻居数均值/std

### 需求: Phase 1 验证集 P/R 总结

Phase 1 训练结束时必须输出最佳阈值下的 Precision/Recall。

#### 场景: 打印最佳验证 P/R

- **当** Phase 1 训练完成，多阈值评估结束后
- **那么** 打印一行 "Best: threshold=X.X P=Y.YYY R=Z.ZZZ"
- **而且** 这个值被写入 Phase 1 checkpoint 的 `phase1_val_best` 字段
